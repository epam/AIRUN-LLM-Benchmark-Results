**Issue**: `Page` and `Users` factories use shared mutable state, which can lead to side effects.

```js
    };
```

**Recommendation**:
Use services with getter/setter methods or return a class instance.

**Improved Code**:
```js
angular.module('cosmo').service('PageService', function(){
    let page = {};
    return {
        getPage: () => angular.copy(page),
        setPage: (data) => { page = angular.copy(data); }
    };
});
```

## 3. 🚀 Performance Optimization

**Improved Code**:

```js
$scope.newerVersion = elements.some(field => {

});
```
