# Technical Documentation: Page Management System

## Overview

The Page Management System is an AngularJS-based module designed to create, edit, and manage web pages within the Cosmo CMS. It provides a comprehensive interface for content management, allowing users to manipulate page content, metadata, publishing status, and associated resources through a RESTful API.

## Architecture

The system follows a Model-View-Controller (MVC) pattern with the following components:

1. **Factories**:
   - `REST`: Provides RESTful API endpoints for various content types
   - `Page`: Stores page data globally across the application
   - `Users`: Maintains current user information

2. **Controllers**:
   - `pageCtrl`: Handles page creation, editing, and management operations

3. **Views**:
   - `page.html`: Provides the user interface for page management

## Features

### REST Factory

The REST factory establishes communication with the backend through RESTful endpoints:

- **Content Management**: Endpoints for blocks, comments, content, revisions, and tags
- **File Management**: Endpoints for files and file tags
- **System Configuration**: Endpoints for menus, modules, sitemaps, themes, and settings
- **User Management**: Endpoints for user data

### Page Factory

Stores and manages page data globally:

- Basic metadata (title, description, URL)
- Content sections (header, subheader, body)
- Publishing information
- Theme configuration
- Timestamps and versioning
- Custom extras and miscellaneous data

### Page Controller

Provides functionality for:

- **Page Creation**: Creating new pages with proper metadata
- **Page Editing**: Modifying existing page content and settings
- **Version Management**: Handling draft versions and revisions
- **URL Management**: Auto-generating and validating URLs
- **Tag Management**: Adding, removing, and auto-completing tags
- **Publishing Control**: Setting publish status (published, draft, scheduled)
- **Local Storage**: Saving drafts to browser storage for recovery

## API Reference

### REST Factory Endpoints

| Endpoint | Description | Methods |
|----------|-------------|---------|
| `api/blocks/:blockID` | Manage content blocks | GET, POST, PUT, DELETE |
| `api/content/:contentID` | Manage page content | GET, POST, PUT, DELETE |
| `api/content/:contentID/revisions/:revisionID` | Manage content revisions | GET, POST, PUT, DELETE |
| `api/content/:contentID/tags/` | Manage content tags | GET, POST, DELETE |
| `api/files/:fileID` | Manage media files | GET, POST, PUT, DELETE |
| `api/themes/:themeID` | Manage themes | GET, POST, PUT, DELETE |
| `api/settings/` | Manage system settings | GET, POST, PUT |
| `api/users/:userID` | Manage user accounts | GET, POST, PUT, DELETE |
