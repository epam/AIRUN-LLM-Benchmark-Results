### **4. Page Controller (`pageCtrl.js`) and Template (`page.html`)**

#### **Overview**

The `pageCtrl` controller and `page.html` template work together to handle the creation and editing of pages in the Cosmo application. They provide a user interface for managing page content, metadata, and publishing settings.

#### **Features**

-   **Page Creation and Editing:** Allows users to create new pages or edit existing ones.
-   **Form Handling:** Manages form inputs for page title, description, URL, tags, type, and publishing settings.
-   **Local Storage:** Uses local storage to save unsaved changes, allowing users to recover their work in case of interruptions.
-   **Tag Autocomplete:** Provides tag suggestions as the user types.
-   **Page Duplication:** Allows users to duplicate existing pages.
-   **Page Deletion:** Enables users to delete pages.
-   **Scheduled Publishing:** Supports scheduling pages to be published at a future date and time.
-   **Revision Handling:** Saves page revisions and allows reverting to previous versions.
-   **Accessibility:** Uses ARIA attributes and roles to enhance accessibility.
