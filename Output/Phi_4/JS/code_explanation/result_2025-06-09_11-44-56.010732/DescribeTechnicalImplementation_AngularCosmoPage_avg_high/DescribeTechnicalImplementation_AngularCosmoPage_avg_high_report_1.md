                $scope.newerVersion = true;
        });

                        extra: Page.extras[key]

# Technical Documentation for Angular.js Application Components

## Component Overview

### Introduction
This documentation provides a comprehensive overview of the Angular.js components used in a web application designed to manage and edit pages. The application leverages Angular.js to create a dynamic and responsive user interface, allowing users to create, edit, and manage content efficiently.

### Key Functionality and Architectural Role
The application consists of several key components:
- **REST Factory**: Handles communication with the RESTful backend.
- **Page Factory**: Stores page-related variables globally.
- **Users Factory**: Manages data about the current user.
- **Page Controller**: Manages page creation, editing, and deletion.

These components work together to provide a seamless content management experience, integrating with a backend API to perform CRUD operations on pages and related data.

The components are part of the `cosmo` Angular.js module, which serves as the core of the application. They interact with each other and with the backend to provide a cohesive user experience.
