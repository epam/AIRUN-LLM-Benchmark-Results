# Technical Documentation for Angular Page Management Application

## Overview

This Angular-based application is designed to manage pages, users, and content through a RESTful backend. The application provides a user-friendly interface for creating, editing, and deleting pages, as well as managing associated metadata such as tags, descriptions, and publishing schedules. The application leverages Angular's `$resource` service to interact with the backend API and provides a modular structure with factories and controllers to handle different aspects of the application.

Key components include:
- **REST Factory**: Handles communication with the backend API.

- **Users Factory**: Stores user-related data globally.
- **Page Controller**: Manages the creation and editing of pages.
