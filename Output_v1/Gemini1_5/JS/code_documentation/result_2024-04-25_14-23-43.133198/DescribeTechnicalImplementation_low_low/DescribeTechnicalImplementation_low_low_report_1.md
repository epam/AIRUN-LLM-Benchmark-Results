### Overview

The `NavBar` component is a React component that provides a navigation bar for the PodBaby application. It offers a consistent and user-friendly way for users to navigate between different sections of the application, such as browsing podcasts, searching for content, and managing their subscriptions. The component adapts its content based on the user's authentication status, displaying relevant options for logged-in and logged-out users.

### Features

*   **Navigation Links:** Provides links to key sections of the application, including "New episodes," "Browse," "Search," and "Recommended."

*   **Authentication-Based Rendering:** Adapts the displayed navigation options based on whether the user is logged in or not.
*   **Add Channel Functionality:** For logged-in users, offers an option to add new podcast feeds to their subscriptions.
